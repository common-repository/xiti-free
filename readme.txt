Xiti free allows the automatic tagging of pages generated by WordPress.

== Description ==

This plugin generate the appriopriate tag on all pages for use with Xiti with a free account.


== Installation ==

1. Download the file into the plug-in folder (./wp-content/plugins/).
2. Activate the plug-in in the 'Extensions' menu in WordPress.
3. Configure the plug-in in the sub-menu 'Xiti Free', in the 'Tools' section:
